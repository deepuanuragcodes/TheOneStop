# OneStop — Digital Growth Agency

A clean, modern agency website built for creators and businesses.

## Pages

- **Home** — `index.html`
- **Services** — `services.html`
- **Portfolio** — `portfolio.html`
- **About** — `about.html`
- **Contact** — `contact.html`

## Files

| File | Purpose |
|------|---------|
| `index.html` | Home page |
| `services.html` | Services listing |
| `portfolio.html` | Portfolio gallery |
| `about.html` | About page |
| `contact.html` | Contact / booking |
| `styles.css` | All styles (shared) |
| `shared.js` | Shared data, cart, checkout logic |

## Deploy on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Hit **Save** — your site will be live at:
   `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

## URL Routing

Each page has its own URL:
- `/` or `/index.html` → Home
- `/services.html` → Services
- `/portfolio.html` → Portfolio
- `/about.html` → About
- `/contact.html` → Contact

No extra config needed — GitHub Pages serves static HTML natively.
